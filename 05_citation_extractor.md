# Module 05 – Citation Extractor

## Purpose
Extract only explicit citations from provided text.

---

## Valid Sources
- References section
- In-text citations: parenthetical or numeric

---

## Rules
- Preserve exact citation strings
- Capture IDs only if clearly included
- No guessing or reconstruction

---

## Insufficiency Handling
- If no citations exist → warning
- If references only partially defined → warning

---

## Output
- List of extracted citations
- Any associated citation warnings
